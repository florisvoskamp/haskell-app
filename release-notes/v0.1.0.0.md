# BudgetFlow v0.1.0.0

## Release Name

`BudgetFlow v0.1.0.0`

## Overview

This release packages the full `haskell-app` application as command line binaries for macOS, Windows, Linux, and Arch Linux.
The project combines two parts:

1. The original recursion assignment utilities that run in the default mode.
2. `BudgetFlow`, a personal budget simulation CLI built in Haskell.

## Included Application Features

### BudgetFlow CLI

- Run a baseline monthly budget simulation from a TOML configuration file.
- Compare a baseline simulation against a scenario file with additional events.
- Run a Monte Carlo stress test with seeded random variable expenses.
- Print end balances, warnings, scenario differences, and percentile based risk output.

### Functional Design Highlights

- Pure simulation core separated from IO.
- Immutable month by month state transitions.
- Higher order functions such as `foldl`, `scanl`, `map`, and `iterate`.
- Pattern matching for events, rules, and distributions.
- Lazy timeline generation for simulation over multiple months.
- A custom TOML parser tailored to the application configuration format.

### Recursion Exercises

- Greatest common divisor
- Palindrome check
- Digit sum
- Reverse list
- Binary printer

## Usage

```bash
./haskell-app run test.toml
./haskell-app scenario test.toml scenario.toml
./haskell-app stress test.toml
```

## Notes

- Amounts in the configuration files are stored in cents.
- Monte Carlo output is reproducible when the same seed and input data are used.
- The default application mode also runs the recursion assignment examples before the BudgetFlow demo flow.
